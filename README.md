# 🚀 Todo Task Manager API

A professional and secure Todo Task Manager REST API built using **Django REST Framework (DRF)** with **JWT Authentication**.

This project allows users to manage their personal tasks with authentication, filtering, pagination, search, ordering, and secure user-based access.

---

# ✨ Features

## 🔐 Authentication

* User Registration
* User Login
* JWT Authentication
* Refresh Access Token
* Protected APIs

---

## ✅ Task Management

* Create Task
* Update Task
* Delete Task
* Mark Complete / Incomplete
* Task Priority System
* Due Date Management

---

## ⚡ Advanced Features

* Search Tasks
* Filter Tasks
* Ordering
* Pagination
* User-specific Task Access
* Secure APIs

---

# 🛠️ Tech Stack

* Python
* Django
* Django REST Framework
* JWT Authentication
* SQLite3
